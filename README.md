## üöÄ Quick Start

### Prerequisites
- Python 3.8 or higher
- `pip` package manager
- Determination and discipline

### Installation

bash
---------------------------------------------------------------------------
# Clone the repository
git clone https://github.com/tyler/tux-boot-camp.git
cd tux-boot-camp

# Install dependencies
pip install aiohttp

# Generate resource files
python generate_resources.py

# Launch the boot camp
python TuxCodeBootCamp.py
---------------------------------------------------------------------------

### First Launch

1. **Enroll** - Enter your name (your recruit identifier)
2. **Choose Your Weapon** - Select a programming language
3. **Take a Challenge** - Accept your first mission
4. **Code** - Complete the challenge in your favorite editor
5. **Submit** - Let Sergeant Tux review your code
6. **Improve** - Learn from feedback and level up

## üí° How It Works

### 1. **Choose Your Path**
Select a programming language from our extensive arsenal. Each language includes:
- Difficulty rating
- Use cases
- Learning resources
- Sample code
- Sergeant Tux's expert take

### 2. **Accept Missions**
Each language has three challenge levels:
- **BEGINNER DRILL** - Basic fundamentals
- **INTERMEDIATE MISSION** - Practical applications
- **ADVANCED OPERATION** - Complex systems

### 3. **Code & Submit**
- Automatic challenge file generation
- Pre-loaded templates with instructions
- Submit your code for AI-powered analysis

### 4. **Get Feedback**
Sergeant Tux analyzes your code and responds with:
- **Correctness** - Does it work?
- **Completeness** - Did you meet all requirements?
- **Quality Score** - How clean is your code?
- **Emotional Response** - From "EXCEPTIONAL!" to "WHAT IS THIS?!"

### 5. **Level Up**
Track your motivation level as you:
- Commit to languages
- Complete challenges
- Write quality code
- Build discipline

### Key Principles
- ‚úÖ **Separation of Concerns** - All strings externalized to JSON
